# Обзор репозиториев Localzet на GitHub

Добро пожаловать на официальную страницу обзора репозиториев Localzet на GitHub. Здесь вы найдете полный список моих проектов, сгруппированных по категориям.

## Фреймворки для разработки

### [Triangle-Vue](https://github.com/localzet/Triangle)
Triangle — это универсальный фронтенд фреймворк на базе Nuxt (Vue), предназначенный для создания веб-приложений. Он использует архитектуру MVC (Model-View-Controller) и легко интегрируется с современными инструментами, такими как Tailwind CSS и TypeScript. Он идеально подходит для разработчиков, желающих создавать динамичные, адаптивные веб-приложения.

### [Triangle-Web](https://github.com/Triangle-org/Web)
Triangle-Web — это универсальный фреймворк, написанный на PHP предназначенный для создания высокопроизводительных веб-приложений. Он использует Localzet Server и поддерживает многие современные технологии.

### [Triangle-WS](https://github.com/Triangle-org/WS)
Triangle-WS — это фреймворк на основе WebSocket, написанный на PHP предназначенный для создания высокопроизводительных веб-приложений с использованием асинхронного подхода. Он использует Localzet Server и поддерживает многие современные технологии.

## Библиотеки для серверного программирования

### [Localzet Server](https://github.com/localzet/Server)
Localzet Server — это асинхронный event-based сервер для PHP, обеспечивающий высокую производительность и масштабируемость с использованием современных технологий и стандартов.

### [Tunnel](https://github.com/localzet/Tunnel)
Localzet Tunnel — это библиотека кросс-серверных туннелей. Она предоставляет функциональность для создания защищенных туннелей между серверами, облегчая эффективную и защищенную кросс-серверную коммуникацию.

### [HTTP Proxy](https://github.com/localzet/Proxy)
Localzet HTTP Proxy — это HTTP-прокси-сервер. Он предоставляет функциональность для эффективной и безопасной переадресации запросов.

### [SOCKS5 Proxy](https://github.com/localzet/Socks5)
Localzet SOCKS5 Proxy — это SOCKS5-прокси-сервер. Он предоставляет функциональность для эффективной и безопасной переадресации запросов.

### [Storage](https://github.com/localzet/Storage)
Localzet Storage — это аналог горячего хранилища на PHP для эффективного хранения и управления данными. 

### [MQTT](https://github.com/localzet/MQTT)
MQTT — это сервер для работы с протоколом MQTT, обеспечивающая функциональность для обмена сообщениями в системе "публикация-подписка".

## Общие библиотеки PHP

### [Events](https://github.com/localzet/Events)
Events — это надежная библиотека управления событиями для PHP-приложений. Она позволяет легко регистрировать, вызывать и управлять событиями с поддержкой обработчиков событий с префиксами.

### [WebAnalyzer](https://github.com/localzet/WebAnalyzer)
WebAnalyzer — это инструмент веб-анализа, анализирующий посетителей веб-сайта, получая информацию об их устройстве, операционной системе, браузере и т.д.

### [Console](https://github.com/localzet/Console)
Console — это инструмент командной строки для выполнения различных задач и управления проектами. Написан на PHP, наследует Symfony Console.

### [HttpClient](https://github.com/localzet/HttpClient)
HttpClient — это библиотека для асинхронного выполнения HTTP-запросов, создана с использованием технологий Localzet Server для обеспечения эффективной и надежной связи.

### [ZConf](https://github.com/localzet/ZConf)
ZСonf — это PHP-библиотека, разработанная для работы с языком конфигурирования Zorin (ZCONF), предоставляющая возможности, аналогичные TOML или INI.

### [WebAuthn](https://github.com/localzet/WebAuthn)
Localzet WebAuthn — это высокопроизводительная библиотека PHP для работы с WebAuthn, стандартом веб-аутентификации, разработанным W3C. Библиотека обеспечивает безопасную и надежную аутентификацию пользователей.

### [SocketIO](https://github.com/localzet/SocketIO)
Localzet Socket — это библиотека PHP SocketIO, разработанная Localzet. Она предоставляет функциональность для двусторонней событийной коммуникации в реальном времени.

### [JWT](https://github.com/localzet/JWT)
Localzet JWT — это библиотека для работы с JSON Web Tokens (JWT). Она использует современные технологии и стандарты для создания, проверки и обработки JWT.

### [LWT](https://github.com/localzet/LWT)
Localzet LWT — это библиотека для работы с Localzet Web Tokens, основанными на JWT (JSON Web Tokens). Она предоставляет функциональность для создания, проверки и обработки токенов с использованием современных технологий.

### [OAuth](https://github.com/localzet/OAuth)
Localzet OAuth — мощная библиотека для OAuth-аутентификации, поддерживающая различных популярных провайдеров, таких как Google, Facebook, Twitter и Apple (50+ провайдеров).

### [PSR](https://github.com/localzet/PSR)
Localzet PSR — это компонент для проектов, предоставляющий набор общих интерфейсов для PHP. Он соответствует рекомендациям PHP Standard Recommendations (PSR).

### [PSR-7](https://github.com/localzet/PSR-7)
Localzet PSR-7 — это библиотека, предоставляющая интерфейсы HTTP-сообщений. Она соответствует стандарту PSR-7 для интерфейсов HTTP-сообщений в PHP.

## Проекты XRay

### [X-UI](https://github.com/localzet/X-UI)
X-UI — это легковесная, удобная веб-панель для управления конфигурациями Xray. Идеально подходит для личного использования, упрощает процесс управления серверными конфигурациями, делая его доступным для всех.

### [X-Tun](https://github.com/localzet/X-Tun)
X-Tun — это универсальная PHP-библиотека, предназначенная для создания и управления безопасными туннельными соединениями. Она обеспечивает эффективную, зашифрованную связь через сети, гарантируя конфиденциальность и безопасность данных.

### [X-Xin](https://github.com/localzet/X-Xin)
X-Xin — это клиент XRay для Windows, предназначенный для обеспечения бесшовной интеграции и управления сервисами XRay на платформах Windows. Он поддерживает легкое переключение между различными серверными окружениями, улучшая пользовательский опыт и продуктивность.

### [XRay-gRPC](https://github.com/localzet/XRay-gRPC)
XRay-gRPC — это мощная PHP-библиотека для управления и взаимодействия с XRay через gRPC. Она упрощает процесс отправки и получения данных, что делает ее идеальной для разработчиков, ищущих эффективного взаимодействия с сервисами XRay.

### [XRay-gRPC-PHP](https://github.com/localzet/XRay-gRPC-PHP)
XRay-gRPC-PHP — это универсальная PHP-библиотека, предназначенная для облегчения взаимодействия с XRay с использованием gRPC. Эта библиотека упрощает отправку и получение данных через протоколы gRPC, обеспечивая эффективное взаимодействие и управление сервисами XRay.

## Проекты Shadowsocks

### [Shadowsocks-Rust](https://github.com/localzet/Shadowsocks-Rust)
Shadowsocks-Rust — это реализация протокола Shadowsocks на языке Rust, предназначенная для обеспечения безопасных и эффективных прокси-сервисов. Она предлагает надежное шифрование и идеально подходит для обхода интернет-цензуры и обеспечения конфиденциальности.

### [Shadowsocks-PHP](https://github.com/localzet/Shadowsocks-PHP)
Shadowsocks-PHP — это реализация протокола Shadowsocks на языке PHP, предназначенная для обеспечения безопасных и эффективных прокси-сервисов. Она предлагает надежное шифрование и идеально подходит для обхода интернет-цензуры и обеспечения конфиденциальности.

### [Shadowsocks-Android](https://github.com/localzet/Shadowsocks-Android)
Shadowsocks-Android — это клиент протокола Shadowsocks для Android, предназначенный для обеспечения безопасных и эффективных прокси-сервисов на устройствах Android. Он обеспечивает конфиденциальность и помогает обойти интернет-цензуру с помощью надежного шифрования.

### [Shadowsocks-Windows](https://github.com/localzet/Shadowsocks-Windows)
Shadowsocks-Windows — это клиент протокола Shadowsocks для Windows, предназначенный для обеспечения безопасных и эффективных прокси-сервисов на устройствах Windows. Он обеспечивает конфиденциальность и помогает обойти интернет-цензуру с помощью надежного шифрования.

### [Shadowsocks-Crypto](https://github.com/localzet/Shadowsocks-Crypto)

### [Shadowsocks-Manager-PHP](https://github.com/localzet/Shadowsocks-Manager-PHP)
Shadowsocks-Manager-PHP — это инструмент для управления серверами Shadowsocks, разработанный для упрощения администрирования сервера и улучшения пользовательского опыта. Он предоставляет функции для многопользовательского управления, контроля трафика и простой настройки.

Shadowsocks-Manager-JS — это инструмент для управления серверами Shadowsocks, разработанный для упрощения администрирования сервера и улучшения пользовательского опыта. Он предоставляет функции для многопользовательского управления, контроля трафика и простой настройки.
